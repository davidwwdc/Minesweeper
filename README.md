# Minesweeper - Python

This is a text‑based Minesweeper game developed in Python. The game runs in the console and allows players to uncover cells, mark flags, and attempt to clear the field without detonating mines. It enforces classic Minesweeper rules and offers a simple interactive experience.

---

## Features

- Play Minesweeper in the console  
- Randomly generated minefield  
- Uncover cells and reveal adjacent counts  
- Flag suspected mines  
- Classic Minesweeper gameplay mechanics  

---

## Requirements

- Python 3.x  

---

## Installation and Setup

1. Clone the repository:

```bash
git clone https://github.com/davidwwdc/Minesweeper.git
cd Minesweeper
```

2. Run the game:

```bash
python main.py
```

> Replace `main.py` with the actual entry file name if different.

---

## How to Play

- The game displays a grid of hidden cells  
- Enter coordinates to uncover a cell  
- Numbers indicate how many adjacent mines there are  
- Flag cells you believe contain mines  
- Clear all non‑mine cells to win
